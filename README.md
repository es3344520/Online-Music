R2音乐库，把音乐存储在cloudflare R2  api请求接口部署在Vercel  

为了安全且快速调用接口，Vercel最好是有自己的域名，R2建议配置自定义域名 

Vercel相关的环境变量命名可以在api/music.js里面查看。前4个令牌是必须的，R2域名自定义之后可以填写第五个【非必须】
 
环境变量命名之后，配置好R2的相关密钥，就可以调用此api接口

此接口需搭配支持R2音乐库的播放器使用，方可实现音乐播放功能

注意：配置R2的时候不要开启公开访问，为了防止链接盗用，每一次请求都携带签名，且有过期时间，默认有效10分钟，可以在api/music.js进行配置

注意：部署项目之后，尽快删除Vercel分配的Domains域名，这个会暴露api接口， 从而导致接口被滥用

注意：暴露api接口之后，即便没有R2相关密钥，也可以通过api接口的调用，获取R2里面的文件
